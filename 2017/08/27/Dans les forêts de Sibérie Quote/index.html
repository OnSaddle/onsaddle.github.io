Quotes from Dans les forêts de Sibérie 金句三則

Author: Wolf Date: August 27, 2017  16:06:24 Category: Skiff

我們是唯一應該對自己蕭索的生命負責的人。世界因我們的乏味而灰暗。

  • 西爾萬·泰松

如果你覺得你的日常生活很貧乏，你不要抱怨它。還是怨恨你自己吧，怨你還不夠做一個詩人來呼喚生活中的寶藏。

  • 里爾克

我們向世界說話的語氣便是它對我們使用的語氣。付出最好的，收穫最好的。

  • 約翰·巴勒斯

摘自 Dans les forêts de Sibérie 《在西伯利亞森林中》
